title: std:render
link_as: keyword
--

A utility template intended to be used for rendering deferred renders from other templates.

The template is just a single [node:plugin/templates/functions/language#render_deferred] template function.

<pre>
    render(.)
</pre>

Use it to render the result of [node:plugin/interface/template#deferredRender].

<pre>language=javascript
var deferred = P.template("example").deferredRender({ ... });
var html = P.template("std:render").render(deferred);
</pre>
