title: Transitions
long_title: Transitions interface
link_as: keyword
--

The @Transitions@ object stores information about the currently available transitions for a [node:dev/standard-plugin/workflow/interfaces/instance], available as @M.transitions@.



h2. Interface


h3(property). list

An array of all valid transitions, represented by [node:dev/standard-plugin/workflow/interfaces/transition] objects.


h3(property). empty

@true@ if there are no valid transitions.


h3(function). has(transition)

Returns @true@ if the named @transition@ is valid for this state.


h3(function). properties(transition)

Returns a [node:dev/standard-plugin/workflow/interfaces/transition] with information about the named @transition@, or @undefined@ if this transition is not valid.
