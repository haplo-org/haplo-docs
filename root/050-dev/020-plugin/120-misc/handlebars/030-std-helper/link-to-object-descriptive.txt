title: std:link_to_object_descriptive
link_as: keyword
--

This helper renders an @<a>@ tag to link to a [node:dev/plugin/interface/store-object], generating a link URL and the HTML escaped full *descriptive* title of the object.

<pre>
    {{std:link_to_object_descriptive object}}
</pre>

@object@ is the name of the property in the view which contains the object.
