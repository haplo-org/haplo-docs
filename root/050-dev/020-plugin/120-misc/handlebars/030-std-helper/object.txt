title: std:object
link_as: keyword
--

This helper renders a [node:dev/plugin/interface/store-object], taking into account it's type and the style requested. The HTML generated will depend on the style requested and the 'render type' set in the schema for this type of object.

<pre>
    {{std:object object "style"}}
</pre>

@object@ is the name of the property in the view which contains the object to render.

@"style"@ is an optional string literal specifying the style in which to render the object. See [node:dev/plugin/templates/std-template/object] for the possible styles.
