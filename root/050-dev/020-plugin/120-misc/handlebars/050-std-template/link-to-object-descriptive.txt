This template renders an @<a>@ tag to link to a [node:dev/plugin/interface/store-object], generating a link URL and the HTML escaped full *descriptive* title of the object.

There is also a [node:dev/plugin/misc/handlebars/std-helper/link-to-object-descriptive] helper, which may be easier to use.


h2. View

h3(key). object

The [node:dev/plugin/interface/store-object] to link to.
