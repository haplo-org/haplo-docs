title: O.group()
link_as: keyword
--

Look up a group, and return an object implementing the [node:dev/plugin/interface/security-principal] interface.

h3(function). O.group(groupId)

Look up the group with this group ID. Group IDs are exposed as schema [node:dev/plugin/schema/schema-global/group] constants.

An exception will be thrown if the group does not exist.
